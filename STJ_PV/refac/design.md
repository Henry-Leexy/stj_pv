# Subtropical jet finder

## Requirements

### Input Data
- Daily / monthly / sesasonal?
- Isobaric / isentropic
- PV exists / PV to be generated

### Output Data
- Jet position function of (hemisphere, [longitude])
    - Latitude
    - Theta

### Flexible parameters
- PV contour level
- Degree of polynomial interpolation
- Minimum latitude for fit
